## 1. Register firebase credentials
ChattyCloud need to register your firebase credential to send fcm messages to your chat members at dashboard

That's it. now you can receive push notifications 

![dashboard-firebase-credentials](/img/docs/dashboard-fcm-credentials.png)

## 2. How to show missed count badge

```typescript
import Chatty from "chatty-client";

/**
 * @returns {Promise<{total: number, group: Array<{name: string, count: number}>}>}
 */
async function getMissedCount() {
  const data = await Chatty.getMissedCount();
  return data;
}
```
