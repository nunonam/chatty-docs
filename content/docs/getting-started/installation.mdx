## Client SDK installation

with npm
```npm
npm install chatty-client
```

with yarn
```yarn
yarn add chatty-client
```

<InfoBlock type="note">
  this sdk does not need to pod install
</InfoBlock>


## Cloud Setup

It's needed to get API key to use sdk at client frontend. To do this, Create new application at [`dashboard`](https://devdashboard.chatty-cloud.com)

If your app is created, goto menu apps and then you can get API key


![API key at dashboard app tab](/img/docs/dashboard-apikey.png)
